# queryparent
JavaScript's missing `queryParent` function.

Takes two arguments:

1. A `selector`.
2. A `parentNode` of that selector.

Given element `.bar` with parent element `.foo`, you would do:

```javascript
queryParent('.bar', '.foo');
// => Returns `.bar`.
```
